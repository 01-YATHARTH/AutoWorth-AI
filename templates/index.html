About This Project

AutoWorth AI is a smart car price predictor built using a machine learning model trained on over 8,000 real-world car listings. Powered by Flask on the backend and designed with modern, responsive UI/UX for a seamless experience.

This tool helps users uncover fair market values of used cars based on real data — fast, transparent, and accurate.
